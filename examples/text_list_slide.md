# Les Fondamentaux du Machine Learning

Le Machine Learning repose sur trois grandes familles d'algorithmes :

- **Apprentissage supervisé**
  - Régression linéaire
  - Forêts aléatoires
  - Réseaux de neurones
- **Apprentissage non supervisé**
  - K-Means
  - DBSCAN
  - Analyse en composantes principales (ACP)
- **Apprentissage par renforcement**
  - Q-Learning
  - Policy Gradient

> « Les données sont le nouveau pétrole. » — Clive Humby
